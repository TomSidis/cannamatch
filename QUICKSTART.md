# 🚀 התחלה מהירה — 3 פקודות

```bash
npm install
cp .env.example .env      # ← פתח וערוך: הכנס מפתח API
npm run dev:full
```

פתח: http://localhost:5173

---

## הערות

- **בלי מפתח API?** הרץ `npm run dev` — הממשק עובד, רק פיצ'רי ה-AI מושבתים.
- **העוזר "צמח" משתמש ב-web search.** ודא שלמפתח ה-API שלך יש גישה לכלי החיפוש (רוב המפתחות כן). אם החיפוש לא עובד, העוזר עדיין יענה — פשוט בלי מידע מהאינטרנט.
- **תמונות תפריט** — הפיצ'ר שולח תמונה ל-Claude vision. עובד עם כל מפתח API תקין.

## פקודות

| פקודה | מה היא עושה |
|---|---|
| `npm run dev` | רק הקליינט (ללא AI) |
| `npm run server` | רק שרת ה-proxy |
| `npm run dev:full` | שניהם יחד (מומלץ לפיתוח) |
| `npm run build` | בונה ל-production (תיקיית dist/) |
| `npm run preview` | תצוגה מקדימה של הבנייה |
