## ה-API endpoints (Node — api/server.js)

| Method | Path | מה הוא עושה |
|---|---|---|
| GET  | `/api/health` | בדיקת חיים |
| GET  | `/api/strains` | שליפת זנים (סינון: type, q) |
| GET  | `/api/dna/:userId` | פרופיל ה-DNA של המשתמש |
| PUT  | `/api/dna/:userId` | עדכון DNA מדיווח (למידה) |
| POST | `/api/dna/:id/checkin` | צ'ק-אין יומי + הודעת חבר |
| GET  | `/api/social/twins/:id` | התאומים הגנטיים + הפיד |
| POST | `/api/parse-menu` | פענוח תפריט (תמונה/טקסט) → מדורג |
| POST | `/api/reviews` | דיווח + עדכון DNA אטומי |
| POST | `/api/claude` | proxy ל-Claude (העוזר "צמח") |

## הזרימה המרכזית

```
תמונת תפריט
   ↓ POST /api/parse-menu
ocr.js (Claude Vision) → שמות מוצרים
   ↓
התאמה fuzzy ב-DB (pg_trgm)
   ↓ אם לא נמצא
fallback.js → ניחוש AI + cache קבוע ב-DB
   ↓
scoring.js → calculateMatchScore (50% גנטיקה / 30% טרפנים+טריגר / 20% סנטימנט)
   ↓
מערך מדורג → ה-UI
```

## הרצה

```bash
# Frontend בלבד (UI עם demo data — הכי מהיר לראות)
npm install
npm run dev          # → localhost:5173

# Frontend + proxy ל-Claude (לעוזר ופענוח תפריט)
cp .env.example .env # הכנס מפתח API
npm run dev:full

# Node backend מלא (דורש PostgreSQL)
npm install pg @anthropic-ai/sdk
# הקם DB עם api/db/schema.sql, ואז:
node api/server.js
```

## מצב נוכחי — מה עובד ומה דמו

**עובד מלא (frontend):** כל הטאבים, ההתאמות, ה-DNA, פענוח תפריט (מקומי), קהילה, יומן, מטבח, התאומים (עם demo data), צ'ק-אין יומי.

**דורש DB חי:** ה-endpoints של `/api/strains`, `/api/dna`, `/api/social/twins`, `/api/reviews` — הקוד מוכן, צריך PostgreSQL פעיל. עד אז הרכיבים נופלים בחן ל-demo data.

**דורש מפתח API:** העוזר "צמח", פענוח תפריט מתמונה (Claude Vision), ניחוש זן לא מוכר.

## הבהרות
- שני backends: **Node** (`api/`) ל-CRUD ול-Claude; **Python** (`backend/`) למנוע המדעי (בייסיאני, pgvector). בחר אחד לפרודקשן — Node פשוט יותר להתחלה.
- **פרטיות:** לפני production — הצפנת PII, consent, מחיקה (GDPR/חוק הגנת הפרטיות).
- **רפואי:** מתאים ומדרג, לא מאבחן. כל ערך טיפולי עם הרופא.

## 4. 🔬 הממצאים המרכזיים מהחקר המעמיק החדש

### האפקט הנלווה — יש עכשיו ראיה קלינית אנושית
מחקר **Johns Hopkins (2024)** הוכיח בניסוי כפול-סמיות שלימונן מפחית את החרדה ש-THC מעורר. זו התשתית המדעית החזקה ביותר לכל הרעיון של קנאמאצ׳ — אבל עדיין יש ספקנות מדעית לגיטימית, ולכן הטון שלנו: "מצפן מבוסס-ראיות, לא ערובה".

### PTSD — מחלוקת שחייבים להציג בכנות
מצד אחד: מחקר ישראלי (77 מטופלים) הראה הקלה בסיוטים. מצד שני: **המועצה הלאומית ל-PTSD בישראל המליצה *נגד* מתן קנאביס ל-PTSD.** המשמעות: קנאמאצ׳ לא יציג קנאביס כטיפול מוכח ל-PTSD — אלא "מטופלים רבים מדווחים הקלה, אך הראיות מתפתחות והדעות חלוקות".

### שושלות מאומתות
- **Wedding Cake** = Triangle Kush × Animal Mints (מאושר מגדל). בקנדה נמכר כ-"Pink Cookies" — דוגמה מושלמת לעיקרון שלנו.
- **Gelato** = Sunset Sherbet × Thin Mint GSC.
- **Runtz** = Zkittlez × Gelato — מדגים שכל הזנים החדשים הם הצלבות של ~6 גנטיקות-על.

## 5. ✅ מה הלאה (המלצות לפי עדיפות)

1. **הטמעת מחלוקת ה-PTSD בקוד** — לרכך את הטון בכרטיס ההתוויה של PTSD (כרגע הוא חיובי מדי לאור העמדה הרשמית הנגדית).
2. **הרחבת מאגר CBD/מאוזן** — רק 31 מתוך 357 זנים אינם עתירי-THC. להוסיף זנים עדינים ולסמן אותם בולט.
3. **לחבר את ה-HTML למוצר** — להפוך את מאגר הגנטיקות למסך חי באפליקציה (כרגע הוא קובץ נפרד).
4. **בדיקת שטח** — עם הרישיון, לסרוק תפריט אמיתי דרך העלאת התמונה.
